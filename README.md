# tikv-client
tikv-client is simple command line program


#install

download in page:https://github.com/lingdor/tikv-client/releases

#demo


```shell
./tikv --pd 127.0.0.1:2379
```



```shell
put xx "good\"123"
```

get key1 xx

cat aa.jpg | rawput key1
